PRIVACY POLICY — RANGEFINDER EXTENSION
Effective date: May 2026  ·  Version 1.0.6

01 — OVERVIEW

STRATEGOS RangeFinder is a browser extension for the game Conflict of Nations (conflictnations.com). It overlays a visual range indicator on the game map to assist with aircraft range planning. It does not automate gameplay, read game data, or interact with the game in any way beyond displaying a visual overlay.

This policy describes what data the extension collects, why, and how it is handled.

02 — DATA COLLECTED

The extension collects the following data for license verification purposes:

  • License key — the access key you enter on first use, used to verify your authorization to use the extension.
  • Device fingerprint — a hashed identifier derived from browser properties (user agent, screen dimensions, language, hardware concurrency, timezone) combined with a randomly generated per-install ID. This is used to bind your license key to your specific browser installation and prevent sharing.

No personally identifiable information (name, email, IP address) is collected by the extension itself.

03 — HOW DATA IS USED

Your license key and device fingerprint are sent to a license validation server (hosted on Google Apps Script) each time you load the game. This server checks:

  • Whether your key is active and authorized
  • Whether the key is being used on the same device it was originally activated on

This data is used solely for license enforcement. It is not used for advertising, analytics, tracking, or any purpose beyond verifying your access.

04 — DATA STORAGE

  • On your device — your license key, device fingerprint, and last validation timestamp are stored in Chrome's extension storage (chrome.storage.local), which is private to the extension and inaccessible to websites or other extensions.
  • On the server — your license key and device fingerprint hash are stored in a private Google Sheet accessible only to the operator. No other data is stored server-side.

05 — DATA SHARING

No data is shared with third parties. The license validation server is operated exclusively by the STRATEGOS developer. Google Apps Script is used as the hosting platform; Google's standard platform terms apply to data in transit.

06 — DATA RETENTION

Server-side data (key + fingerprint) is retained for as long as your license is active. You may request deletion by contacting the operator, at which point your key will be revoked and your record removed from the server. Local extension data is removed when you uninstall the extension.

07 — PERMISSIONS USED

  • storage — stores your license key and validation timestamp locally, extension-private only.
  • script.google.com script.googleusercontent.com — used to contact the license validation server. No other network requests are made.
  • conflictnations.com — the extension only activates on this domain. It does not run on any other website.

08 — YOUR RIGHTS

You may at any time:

  • Request to view what data is stored against your license key
  • Request deletion of your server-side data
  • Uninstall the extension, which removes all local data immediately

09 — CONTACT

For questions about this policy or data deletion requests, contact the operator via the STRATEGOS Discord server or at 3ammy1@gmail.com.

10 — CHANGES TO THIS POLICY

This policy may be updated when the extension is updated. The effective date at the top of this page reflects the most recent revision. Continued use of the extension after an update constitutes acceptance of the revised policy.
